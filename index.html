About APPTEST

APPTEST is a lightweight, open-source tool designed for ethical security researchers to identify web vulnerabilities in bug bounty programs. Inspired by Burp Suite, it offers proxying, crawling, scanning, and more.

Features

  • Intercepting Proxy: Capture and modify HTTP/HTTPS traffic.
  • Crawler: Map websites to find URLs and forms.
  • Scanner: Detect XSS, SQLi, open redirects, and header issues.
  • Intruder: Fuzz parameters for vulnerabilities.
  • Repeater: Resend requests with modifications.
  • Reports: Generate bug bounty-ready Markdown reports.

How to Use

  1. Install Python 3 and dependencies: pip install mitmproxy requests beautifulsoup4 urllib3
  2. Download apptest.py.
  3. Run: python apptest.py example.com
  4. Configure your browser to use the proxy at localhost:8080.
  5. Check apptest_report.md for findings.
  6. Optional: Use --intruder param or --repeater for advanced testing.

Ethical Use

APPTEST is for authorized use only. Only test websites with explicit permission, such as those in bug bounty programs (e.g., HackerOne, Bugcrowd). Unauthorized scanning is illegal and unethical.

Always respect program scopes and verify findings manually before reporting.
